# 🚀 System Insight Dashboard

A real-time system monitoring dashboard built with Node.js.

## 🔥 Features
- Live CPU & Memory tracking
- Process monitoring
- Kill process functionality
- Search & filter processes
- Interactive charts

## 🛠 Tech Stack
- Node.js
- Express
- systeminformation
- Chart.js

## ▶️ Run Locally
npm install  
node server.js  

Open: http://localhost:3000
